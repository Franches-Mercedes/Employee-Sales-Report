# Employee Sales Report

## Overview
This Python program reads employee sales information from a data file and generates a formatted sales report. It calculates each employee’s commission based on sales amount, displays their status and commission earned, and summarizes the total number of full-time and part-time employees along with the total commissions paid.

## Features
- Reads employee data from a file
- Calculates commission based on sales thresholds
- Displays a formatted employee sales report
- Counts full-time and part-time employees
- Calculates total commissions paid

## Language Used
- Python

## Skills Demonstrated
- File handling
- Loops
- Conditional statements
- Variables and accumulators
- Numeric calculations
- Formatted output

## Input File Format
The program reads data from a file named `SalesData.dat`.

Each employee record is stored in 3 lines in this order:
1. Sales amount
2. Employee status
3. Employee name
